# Guild Discord Bot

Бот для Discord-сервера игровой гильдии. Напоминания по расписанию, еженедельные опросы на рейды, приветствие новых участников с выбором класса.

## Возможности

- **Напоминания по cron** — тексты и расписание в `config/reminders.json`, с автоудалением сообщений
- **Опросы на рейды боссов** — создаются в понедельник, итоги публикуются в четверг, очистка в воскресенье
- **Приветствие** — сообщение при входе нового участника; шаблон в `config/welcome.json`
- **Выбор класса** — кнопки в канале приветствия, выдают роль и пишут строку в лог-канал
- **Сводка состава** — живое сообщение с количеством Танков / Хиллеров / Дамаггеров
- **Slash-команды**: `/ping`, `/clear_survey`, `/class`

## Стек

Node.js 20+ · TypeScript · discord.js v14 · cron · dotenv  
Хранилище состояния — JSON-файлы, внешняя БД не нужна.

## Быстрый старт

```bash
cp .env.example .env   # заполнить переменные
npm install
npm run dev
```
